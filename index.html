🐧 Pangolin Deployer

Generate a docker-compose file for headless Pangolin deployment.

This page helps make manual Docker Compose installation easier by generating the required Compose file and initial config for you. It does not replace Pangolin’s first-time setup flow: after deployment, you still complete the dashboard setup using the one-time token printed in the pangolin container logs.

For the complete manual install process, see the Pangolin Docker Compose manual.

Docker Compose File

This compose file is static and ready to use. The versions and configuration will be substituted from your environment file.

First-time setup

            
This generator creates the Compose stack and config files, but the first admin setup is completed through Pangolin itself.

            
After starting the stack, get the one-time setup token from the pangolin container logs and visit:

            
https://<your-dashboard-host>/auth/initial-setup

            
For full details, see the Pangolin Docker Compose manual setup page.
